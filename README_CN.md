## 使用说明

![how_to_use](img/how_to_use.gif)

1. 打开破解器
2. 向hosts文件(`C:\\Windows\\System32\\drivers\\etc\\hosts`)中添加 `127.0.0.1 yinyuezhushou.com`
3. 点击 **点击激活**
4. 随便输入几个数
5. 点击 **激活**
